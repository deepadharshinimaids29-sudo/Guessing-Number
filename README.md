# Guessing-Number
This is a simple Python game where the computer randomly selects a number between 1 and 100, and the player tries to guess it.
When the program starts:

It welcomes the player.

It generates a random secret number using the random module.

It asks the player to guess the number.

After each guess:

If the guess is too low, the program tells the player to try again.

If the guess is too high, it also asks the player to try again.

If the guess is correct, the program congratulates the player and displays the number of attempts taken.

The game continues in a loop until the correct number is guessed.
